# Skills Intelligence Hub

> **让技能主动找到你**

一套全新的企业级 AI 技能管理平台，从"存储分发"升级为"智能技能引擎"。

---

## 项目简介

Skills Intelligence Hub 是一个公司内部的技能中心，用于集中管理、分发和共享 AI 编程工具的技能模板。核心理念是让技能主动找到用户，而非用户被动搜索。

### 核心特性

- **技能市场** — 类似 App Store 的体验，浏览、搜索、一键安装
- **智能助手** — 上下文感知，主动推送相关技能
- **代码仓库** — Git 原生，开发者熟悉的操作方式
- **知识引擎** — 技能可继承、组合、演化

### 支持的 AI 工具

- Claude Code
- Cursor
- GitHub Copilot
- 更多工具持续扩展中...
